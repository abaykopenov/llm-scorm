# Chamilo LMS Auto Installer (Debian / Ubuntu)

Автоматический скрипт установки Chamilo LMS для Debian и Ubuntu.

## Что делает скрипт

-   Устанавливает Apache
-   Устанавливает MariaDB
-   Устанавливает PHP 8.1
-   Скачивает Chamilo
-   Создаёт базу данных
-   Настраивает VirtualHost
-   Включает mod_rewrite
-   Готовит систему к запуску

Chamilo устанавливается в:

/var/www/html/chamilo

Доступ осуществляется по адресу:

http://IP_СЕРВЕРА/chamilo/

------------------------------------------------------------------------

## Поддерживаемые системы

-   Debian 11 / 12
-   Ubuntu 20.04 / 22.04 / 24.04

------------------------------------------------------------------------

## Установка

### 1. Скачать скрипт

Сохраните файл:

install_chamilo.sh

### 2. Сделать исполняемым

chmod +x install_chamilo.sh

### 3. Запустить

sudo ./install_chamilo.sh

------------------------------------------------------------------------

## После установки

Откройте в браузере:

http://IP_СЕРВЕРА/chamilo/

В установщике укажите:

Host: localhost  
Database: chamilo_db  
User: chamilo_user  
Password: StrongPassword123!

------------------------------------------------------------------------

## Параметры по умолчанию

Database name: chamilo_db  
Database user: chamilo_user  
Database password: StrongPassword123!

Рекомендуется изменить пароль в скрипте перед запуском.

------------------------------------------------------------------------

## Возможные проблемы

### 404 на /course/1/about

Проверьте, что включён mod_rewrite и AllowOverride All в VirtualHost.

### Белый экран

Проверьте версию PHP (должна быть 8.1).

### Не открывается с других устройств

Проверьте firewall и сетевые настройки.

------------------------------------------------------------------------

## Авторская схема установки

-   DocumentRoot: /var/www/html
-   Chamilo: /var/www/html/chamilo
-   Apache mod_rewrite: включён
-   База данных: MariaDB
