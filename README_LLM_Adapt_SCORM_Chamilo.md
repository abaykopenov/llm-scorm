# LLM → Adapt → SCORM → Chamilo Pipeline

## Описание проекта

Данный проект реализует автоматизированную цепочку создания e-learning
курсов:

LLM ↓ Adapt JSON Generator ↓ Adapt (Authoring Tool или CLI) ↓ SCORM 1.2
пакет ↓ Chamilo LMS

Цель — автоматически генерировать учебные курсы с помощью LLM, собирать
их в формат SCORM через Adapt и загружать в Chamilo LMS.

------------------------------------------------------------------------

## Архитектура решения

### 1. LLM

Генерирует структурированный JSON с контентом курса: - Заголовки -
Страницы - Блоки - Компоненты - Вопросы (MCQ, True/False и т.д.)

Это НЕ формат Adapt, а промежуточный формат.

------------------------------------------------------------------------

### 2. Adapt JSON Generator

Скрипт преобразует JSON от LLM в структуру Adapt Framework:

-   course.json
-   contentObjects.json
-   articles.json
-   blocks.json
-   components.json

Каждый элемент должен иметь: - уникальный \_id - parentId - корректные
связи

------------------------------------------------------------------------

### 3. Adapt


— Adapt Framework + CLI

-   Поместить JSON в структуру проекта
-   Выполнить сборку (grunt build)
-   Убедиться, что подключён Spoor
-   Упаковать в SCORM

------------------------------------------------------------------------

### 4. SCORM

На выходе должен получиться корректный SCORM 1.2 пакет:

-   imsmanifest.xml
-   index.html
-   папка build/
-   JS API вызовы (LMSInitialize, LMSCommit, LMSFinish)

------------------------------------------------------------------------

### 5. Chamilo LMS

В Chamilo:

Курс → Learning Path → Import SCORM → загрузить ZIP

Chamilo отвечает за: - учёт пользователей - трекинг прохождения -
отчёты - прогресс студентов

------------------------------------------------------------------------

## Требования

-   Debian или Ubuntu
-   Node.js (если используется CLI)
-   Adapt Authoring Tool или Adapt Framework
-   Chamilo LMS
-   PHP 8.1+ (для Chamilo)
-   MariaDB / MySQL

------------------------------------------------------------------------

## Поток автоматизации

1.  LLM генерирует JSON
2.  JSON преобразуется в Adapt формат
3.  Adapt собирает курс
4.  Публикуется SCORM
5.  SCORM загружается в Chamilo

## Назначение

Данная система предназначена для:

-   автоматической генерации обучающих курсов
-   быстрого создания SCORM пакетов
-   интеграции с LMS
-   корпоративного обучения
-   образовательных платформ

------------------------------------------------------------------------

## Ответственность компонентов

LLM — отвечает за контент  
Adapt — отвечает за визуальную структуру и интерактивность  
SCORM — отвечает за стандартизацию  
Chamilo — отвечает за управление обучением и трекинг

------------------------------------------------------------------------

## Итог

Проект реализует полностью автоматизированный конвейер создания и
публикации электронных курсов от генерации контента до загрузки в LMS.
